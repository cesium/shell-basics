# Navigation Commands

----

# Print Working Directory

```bash
$ pwd
/home/username
```

----

# List

|  |  | |
|------|------|---------------|
| **`ls`** | `-a` | (all files)   |
|      | `-l` | (long format) |
|      | `-t` | (by time)     |

----

# Change Directory

|  |  | |
|-----------|------|---------------|
| **`cd PATH`** | `.`  | (current DIR) |
|           | `..` | (parent DIR)  |
|           | `~`  | (home DIR)    |
