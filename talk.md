# Navigation Commands

## Print Working Directory

```bash
$ pwd
/home/username
```
## List

|          |      |               |
|----------|------|---------------|
| **`ls`** | `-a` | (all files)   |
|          | `-l` | (long format) |
|          | `-t` | (by time)     |

## Change Directory

|               |      |               |
|---------------|------|---------------|
| **`cd PATH`** | `.`  | (current DIR) |
|               | `..` | (parent DIR)  |
|               | `~`  | (home DIR)    |


# Create your own

## Make Directory

|             |      |                  |
|-------------|------|------------------|
| **`mkdir`** | `-p` | (create parents) |
|             | `-v` | (verbose)        |

```bash
$ mkdir [OPT] DIR
```

## Create Empty File

```bash
$ touch FILE
```
